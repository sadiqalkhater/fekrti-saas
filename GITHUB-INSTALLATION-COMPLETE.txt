🚀 **الخطوات السريعة للبدء**
═════════════════════════════════════════════════════════════════

【 1️⃣ إنشاء Repository على GitHub 】

1. اذهب إلى https://github.com
2. اضغط "+" → "New repository"
3. اسم المشروع: fekrti-saas
4. اختر Public أو Private
5. ✅ Add README
6. ✅ Add .gitignore (Node)
7. اختر MIT License
8. Create repository

【 2️⃣ استنساخ محلياً 】

git clone https://github.com/sadiqualkhater/fekrti-saas.git
cd fekrti-saas

【 3️⃣ نسخ الملفات المُعدة 】

نسخ جميع الملفات من /mnt/user-data/outputs/ إلى مجلد المشروع

【 4️⃣ تثبيت المكتبات 】

npm install

【 5️⃣ إعداد البيئة 】

cp .env.example .env.local
# ثم عدّل .env.local وأضف البيانات الحقيقية

【 6️⃣ إعداد قاعدة البيانات 】

npx prisma migrate dev --name init

【 7️⃣ رفع إلى GitHub 】

git add .
git commit -m "feat: initial setup of fekrti saas system"
git push origin main

🆘 **الأخطاء الشائعة والحلول**
═════════════════════════════════════════════════════════════════

❌ خطأ: "DATABASE_URL is not set"
✅ الحل: تأكد من وجود .env.local مع DATABASE_URL

❌ خطأ: "Cannot find module @prisma/client"
✅ الحل: اتبع npm install كاملة

❌ خطأ: Git authentication failed
✅ الحل: استخدم SSH key أو Personal Access Token

❌ خطأ: Port 3000 في الاستخدام
✅ الحل: غيّر PORT في البيئة
